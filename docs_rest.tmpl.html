REST API

The HTTP-based REST API is the modern version of the WISEBED SOAP APIs. All currently actively used and developed client applications (e.g., WiseGui, wisebed.js-scripts) are based on the REST API.

The API is being developed as part of the testbed management software Testbed Runtime. Documentation of the API will be made available soon. Until then, please refer to the implementation of the wisebed.js client library to learn about the API.
